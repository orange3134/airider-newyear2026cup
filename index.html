ルール

基本ルール

  • 2グループ対抗戦
  • グループは試合ごとにシャッフル
  • 人数が奇数の場合はCPUレベル7を補充

試合スケジュール

第1試合：シティトライアル チーム戦 / デスマッチ5
第2試合：シティトライアル チーム戦 / VSナイトメア
第3試合：エアライド リレー / ケイビオン
第4試合：エアライド リレー / エアトピア
バラエティ企画：シティ逃走中 特別ルール

ルール詳細

シティトライアル チーム戦

時間：5分 / ライダー：自由


エアライド リレー

  • 参加10名以内なら2チーム
  • 11名以上ならグループを半分に分けて4チーム
  • 足りない分はCPレベル7を補充
  • 設定：ハンデブースト弱い / ハンデスペシャルなし / 敵キャラあり / その他全て初期値

シティ逃走中

  • 1人ターゲットを決めてシティトライアルを行う
  • 全員でターゲットを追いかけ、3分以内にマシン破壊すれば追跡者チーム側の勝ち
  • 逃げ切ればターゲットの勝ち
  • ライダーは全員カービィ（色は自由）
  • ターゲットはマシンから降りて歩くの禁止
  • ターゲットは抽選で決定
  • 2試合実施
